这一个工具请先启动SQL攻击主文件.py，如果你想启动我们其他的工具，你可以输入选项来确定当前目录下有哪些python可执行文件，是可以进行启动并且运行的，后续如果你想使用其他的文件进行攻击，退出主程序文件，运行其他的python文件
